HTML as One of My Core Skills

HTML is a markup language. This means it uses specific "tags" to tell the browser what each piece of text is. It isn't for logic or math; it is for building the skeleton of a document. It is one of my core skills because I understand how to organize content so that it makes sense to the user and the computer.

The Tags I Used and Why

Tag Category Tags I Used Why I used them
Structural header, footer, section I used these to divide my page into logical parts like the top, middle, and bottom.
Text h1, h2, h3, p I used headings (h) for titles and p for my regular sentences.
Lists ul, li I used these to make my interests look like a neat bulleted list.
Tables table, tr, td Since I can't use CSS, I used these to create rows and columns for my layout.
Media/Links a, img I included these tags so I can link to other pages or show pictures.
Forms form, input I used these to create a place where a visitor can type their name.

How to use HTML correctly

To use HTML well, you have to use "semantic" tags. This means choosing a tag because of what the content is, not how it looks. You also have to make sure every tag that opens is also closed properly.
